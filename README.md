# SwimTimes has moved

This project now lives in the **[ha-apps](https://github.com/dgaus/ha-apps)** repository and is no
longer maintained here. This repository is archived (read-only).

**New home:** https://github.com/dgaus/ha-apps — the Swim Times add-on is in
[`/swim-times`](https://github.com/dgaus/ha-apps/tree/main/swim-times).

## Installing in Home Assistant

Add the repository `https://github.com/dgaus/ha-apps` under
**Settings > Add-ons > Add-on Store > ⋮ > Repositories**, then install **Swim Times**.

---

Swim Times tracks pool occupancy and renders a weekly heatmap so you can find the best times to swim.
